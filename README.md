# ToyReact

3 节课实现玩具 React，以能正常运行 [React 官方教程][1]的 `TicTacToe - 井字游戏`的[最终代码][2]为目标。

本项目记录自己的学习过程和最终成果，视频教程源自极客时间-前端训练营·小课的《winter 手把手带你实现 ToyReact 框架》。
